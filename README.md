# ポートフォリオ

## ■ 概要
Vue3 / TypeScript / Supabaseを用いたWebアプリ開発を行っています。

## ■ 制作物

### 古書管理アプリ
在庫管理・販売管理を目的としたWebアプリです。

- 技術：Vue3 / TypeScript / Supabase
- 機能：CRUD、認証、データ管理

### 茶道具管理アプリ
茶道具の記録・管理を行うツールです。

- 技術：Vue3 / TypeScript / Supabase
- 機能：登録、一覧、検索

## ■ 技術スタック
- Vue3
- TypeScript
- Supabase
- HTML / CSS

## ■ コメント
